Database Systems and Management

Synopsis: I was tasked with working with a partner in creating a new database based off of the USCB 2024 Registrar data. We created an ERD in 3rd Normal Form and created mock data for students in 3 different majors including: Information Science and Tech(ISAT), Computational Science(CSCI), and Psychology(PSYCH). There are 10 students in each grade from incoming to alumni for ISAT and CSCI but only 5 students for PSYCH majors leading up to Fall 2024.

Instructions: To learn more about how we created the ERD please download and look at the PNG file. If you have a blank database please download the SQL file and open it in notepad, then copy and paste this into a query that's assigned to a blank database and run the query.

Computer Programming Techniques, Practices, and Tools

Synopsis: This project was about finding a relationship between two bringing a weapon to school and threatening people with a weapon in the Youth Risk Behavioral Surveillance System. I grabbed the data in YRBSS, put it in Microsoft Access, used SQL to filter the data to the variables needed, put that into an excel sheet, used Seaborn and Pandas library in python and created a user friendly data analysis and graph creation.
